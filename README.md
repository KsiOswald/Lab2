\##Задача 1



\##Текст задачи



Получить список из первых цифр чисел, содержащихся в исходном списке



\###Алгоритм решения



На ввод получается количество чисел, затем сами числа.

После для каждого числа находится первая цифра путем деления его на 10 до тех пор, пока оно не станет <10 и вставляется в новый список

Новый список выводится на экран

\##Задача 2



\###Текст задачи



Сколько раз в списке встречается заданное число? 



\###Алгоритм решения



На ввод получается количество чисел, затем сами числа.

После получается искомое число.

Каждое число сравнивается с искомым числом.

Количество совпадений выводится на экран.
